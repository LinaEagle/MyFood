# MyFood

Приложение для учёта съеденной еды с фото.

## Функционал

- **Главный экран** — список записей, сгруппированных по дате (формат: «22 ноября 2022»), новые сверху
- **Pull-to-refresh** — потяните вниз, чтобы обновить список
- **Выделение и удаление** — тапните по карточке, чтобы выделить одну или несколько записей, затем нажмите на корзину для удаления
- **Добавление записи** — кнопка «+» в правом нижнем углу открывает экран добавления:
  - Выбор фото из галереи
  - Название (опционально)
  - Калории (опционально)
  - Дата и время проставляются автоматически
  - Кнопки «Отмена» / «Сохранить» внизу экрана
- **Сохранение между сессиями** — все данные (фото и мета-информация) хранятся локально на устройстве

## Технологии

- **Kotlin Multiplatform** — общая бизнес-логика для Android и iOS
- **Compose Multiplatform** — декларативный UI
- **Material 3** — дизайн-система
- **kotlinx-serialization** — JSON-сериализация
- **Target SDK**: 35 | **Min SDK**: 24

## Сборка

```sh
./gradlew :composeApp:assembleDebug
```
